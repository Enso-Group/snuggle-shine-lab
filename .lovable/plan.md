## מה נבנה

בוט WhatsApp חכם בעברית עם דשבורד ניהול, שמתחבר ל-Whapi.Cloud (טריאל חינמי), עונה כמו בן אדם, זוכר שיחות, יודע לחפש באינטרנט, ויכול לשלוח לקבוצות לפי פקודה מהדשבורד.

---

## רכיבים עיקריים

### 1. דשבורד ניהול (בעברית, RTL)
- **התחברות** — מסך login (Lovable Cloud Auth)
- **עמוד הגדרות** — הדבקת ה-Whapi API Token + Webhook secret + בדיקת חיבור
- **עמוד שיחות** — רשימת כל השיחות (פרטיות + קבוצות), לחיצה פותחת את ההיסטוריה המלאה
- **עמוד "שלח הודעה"** — בחירת קבוצה/איש קשר + טקסט/פרומפט ("חפש 10 כתבות AI ושלח לקבוצה X") + כפתור שליחה
- **עמוד "אישיות הבוט"** — system prompt בעברית שאפשר לערוך (טון, סגנון, מה מותר/אסור)
- **עמוד לוגים** — מה הבוט עשה, מתי, ולמי

### 2. Backend (TanStack server functions + public webhook)
- `/api/public/whapi-webhook` — נקודת קצה שמקבלת כל הודעה נכנסת מ-Whapi
- שמירת ההודעה ב-DB
- שליחה ל-AI (Lovable AI Gateway, מודל Gemini) עם:
  - System prompt בעברית
  - היסטוריית השיחה הרלוונטית (זיכרון)
  - כלי חיפוש web (אם המשתמש שאל משהו עדכני)
- שליחת התשובה חזרה דרך Whapi API

### 3. מסד נתונים (Lovable Cloud)
- `conversations` — כל שיחה (פרטי/קבוצה, שם, מזהה Whapi)
- `messages` — כל הודעה (כיוון, טקסט, זמן, conversation_id)
- `bot_settings` — system prompt, טוקן Whapi, הגדרות
- `commands_log` — פקודות ידניות שנשלחו מהדשבורד
- `user_roles` — מי admin (רק admin יכול לגשת לדשבורד)

### 4. יכולות AI
- **תשובה אנושית** — Gemini 2.5 Flash דרך Lovable AI Gateway, prompt בעברית שמדגיש טון טבעי
- **זיכרון** — נטען עד 50 ההודעות האחרונות של אותה שיחה לתוך הקונטקסט
- **חיפוש web** — כלי (tool calling) שמפעיל חיפוש כשצריך מידע עדכני, ומחזיר תקציר + לינקים
- **פקודות מהדשבורד** — כשאת כותבת "חפש 10 כתבות על AI ושלח לקבוצה X", השרת מריץ חיפוש → מסכם → שולח דרך Whapi

## זרימה (איך זה עובד בפועל)

```text
מישהו כותב לבוט בוואטסאפ
        ↓
Whapi שולח webhook → /api/public/whapi-webhook
        ↓
שומרים הודעה ב-DB
        ↓
טוענים היסטוריה + אישיות בוט
        ↓
Lovable AI (Gemini) → מחליט אם צריך חיפוש web
        ↓
מחזיר תשובה בעברית
        ↓
שולחים חזרה דרך Whapi API → המשתמש מקבל בוואטסאפ
        ↓
הכל מופיע גם בדשבורד בזמן אמת
```

## שלבי בנייה

1. **הפעלת Lovable Cloud** — DB, Auth, Secrets, AI Gateway
2. **סכמת DB** — טבלאות + RLS + user_roles + admin role
3. **דשבורד** — Login, Layout עם RTL, ניווט בעברית
4. **עמוד הגדרות** — שמירת Whapi token כסוד
5. **Webhook ציבורי** — קבלת הודעות מ-Whapi + שמירה
6. **AI Brain** — server function שמפיק תשובה (היסטוריה + system prompt + web search tool)
7. **שליחה דרך Whapi** — wrapper לקריאות API
8. **עמוד שיחות** — רשימה + צ'אט מלא בזמן אמת (Realtime)
9. **עמוד "שלח הודעה"** — שליחה ידנית/חכמה לקבוצה
10. **עמוד אישיות** — עריכת system prompt
11. **בדיקה מקצה לקצה** — אחרי שתביאי מספר + טוקן

## מה תצטרכי לתת לי בהמשך

1. **API Token מ-Whapi.Cloud** (אחרי שתרשמי + תסרקי QR עם המספר החדש)
2. **Webhook URL** — אני אגיד לך מה להדביק ב-Whapi (משהו כמו `https://your-app.lovable.app/api/public/whapi-webhook`)
3. **אישיות הבוט** — איך את רוצה שידבר? (רשמי/חברי/מצחיק?) — אפשר גם לערוך אח"כ בדשבורד
